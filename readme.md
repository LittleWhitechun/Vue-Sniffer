# Vue-Sniffer
## 前端

前端使用vue+elementui，代码存储在frontend中

使用前需要npm install安装依赖包

## 后端

后端使用Django架构

```
安装依赖后，启动项目
sniffer下 python manage.py runserver 8000 //启动Django
frontend下 npm run serve //启动前端vue项目
```
